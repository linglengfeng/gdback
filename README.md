    golang 1.21.0
    mysql 8.0.33

    ./logs log文件  
    ./pkg 能单独提出来使用的包  
        crypto aes加解密  
        jwt josn web token  
        logger 日志  
        myutil util  
    ./sql sql文件  
    src 源文件
